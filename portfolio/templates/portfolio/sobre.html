Sobre esta Aplicação

Arquitetura MVT do Django

Esta aplicação segue o padrão MVT (Model-View-Template) do Django:

  • Model : define a estrutura dos dados e interage com a base de dados
  • View : recebe os pedidos HTTP, vai buscar os dados ao Model e passa-os ao Template
  • Template : recebe os dados da View e renderiza o HTML final que é enviado ao browser

Modelação

A aplicação tem os seguintes modelos: Licenciatura, UnidadeCurricular, Projeto, Tecnologia, TipoTecnologia, TFC, Competencia, Formacao, MakingOf, Evento.
